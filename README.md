# JavaFX Login Dashboard

A JavaFX desktop application demonstrating MVC architecture with a login screen and dashboard interface.

## Features
- Login screen with CSV-based user authentication
- Dashboard screen displayed after successful login
- MVC separation of concerns
- JavaFX UI built using FXML

## Technologies Used
- Java
- JavaFX
- Maven
- FXML

## Project Structure
- `model` – Core domain classes (User, Person, Ship, AidShip)
- `controller` – JavaFX controllers
- `resources/layouts` – FXML UI layouts
- `resources/data` – CSV data sources

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Ensure JavaFX is configured
4. Run `Launcher.java`

## Academic Context
Originally developed as part of an Application Programming course at UTSA and adapted for portfolio use.
